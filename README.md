# RedMind AI

RedMind AI is a basic web-based project developed during a hackathon. It is designed to analyze source code and highlight simple security issues and provide suggestions.

## Features

* Analyze code for basic issues like:

  * Use of `eval()`
  * Hardcoded passwords
  * Simple SQL injection patterns
* Display a risk score based on detected issues
* Provide basic suggestions for improving code
* Optional attack simulation to show possible risks
* User login and signup system

## Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Python (FastAPI)
* Database: SQLite
* Tools: Antigravity, AI tools (used for assistance during development)
